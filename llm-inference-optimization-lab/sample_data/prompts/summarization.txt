Summarize the following technical concept in 2-3 sentences: Speculative decoding is an inference optimization technique where a smaller, faster draft model proposes multiple candidate tokens that are then verified in parallel by the larger target model. The key insight is that verification of K tokens by the target model can be done in a single forward pass (similar cost to generating one token), making it possible to generate multiple tokens per target model invocation. The speedup depends on the acceptance rate — how often the draft model's predictions match what the target model would have generated. When the draft model is well-aligned with the target model, acceptance rates of 70-90% are achievable, leading to 2-3x latency improvements without any quality degradation.
